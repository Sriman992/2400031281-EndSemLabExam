# Vite + React (fdexam)

This is a minimal Vite + React starter created in your workspace.

Quick start (Windows PowerShell):

```powershell
cd "C:\Users\dashs\OneDrive\Desktop\FDEXAM"
npm install
npm run dev
```

Open http://localhost:5173 in your browser. Build with `npm run build` and preview with `npm run preview`.
